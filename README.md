Это веб-интерфейс для администраторов и учителей.
Их данные захешированны.
Администратор управляет ролями, группами, кафедрами.
Учителя могут просматривать результаты викторин, и оценивать, и изменять статусы заявок на конкурсы, но только у тех студентов с тойже кафедры. 
Просматривать, удалять, создовать, редоктировать викторины и конкурсы.
